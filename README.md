 Tax Calculator GUI (Java)

 Description

This project is a **GUI-based Tax Calculator** built using Java Swing.
It allows users to input their annual income and calculates the tax based on predefined income slabs.
It also includes a **Reset button** to clear inputs and results.

---

 Features

* Simple and user-friendly GUI
* Input field for annual income
* Tax calculation using slab system
* Reset button to clear data
* Error handling for invalid input

---

 Tax Slabs Used

| Income Range           | Tax Rate |
| ---------------------- | -------- |
| ₹0 – ₹2,50,000         | 0%       |
| ₹2,50,001 – ₹5,00,000  | 5%       |
| ₹5,00,001 – ₹10,00,000 | 10%      |
| Above ₹10,00,000       | 20%      |

## ▶️ How to Run

1 Save the file as:

   ```
   TaxCalculatorGUI.java
   ```

2 Compile the program:

   ```bash
   javac TaxCalculatorGUI.java
   ```

3 Run the program:

   ```bash
   java TaxCalculatorGUI
   ```

How It Works

* User enters annual income
* Clicks **Calculate Tax** button
* Tax is calculated based on slabs
* Result is displayed on screen
* **Reset button** clears input and output
